Helm examples for JBoss EAP

Usage with helm

Helm must be installed to use the charts. Please refer to Helm's documentation to get started.

Once Helm has been set up correctly, add the repo as follows: 

        helm repo add jboss-eap-examples https://jmesnil.github.io/helm-charts

If you had already added this repo earlier, run helm repo update to retrieve the latest versions of the packages. You can then run helm search repo jboss-eap-examples to see the charts.

To install the 

<chart-name>
pre> chart:

    
      
        helm install my-<chart-name> jboss-eap-examples/<chart-name>
      

    
    
    
To uninstall the chart:
    
      
        
          helm delete my-<chart-name>  
        

      

    
Integration with OpenShift Web Console


    
The repo can be added to the OpenShift Web Console as a ProjectHelmChartRepository resource to install its charts
    from the OpenShift Web Console.


    
The resource definition is:

    
      
        apiVersion: helm.openshift.io/v1beta1
        kind: ProjectHelmChartRepository
        metadata:
          name: jboss-eap-up-and-running
        spec:
          connectionConfig:
            url: https://jmesnil.github.io/helm-charts/
          name: Examples for JBoss EAP
          description: >-
            This Helm Chart repository contains examples to deploy JBoss EAP
            applications on OpenShift        
      

    

    
You can add it to OpenShift Web Console by pasting it as a new YAML resource or by running the command:


    
      
        oc apply -f https://jmesnil.github.io/helm-charts/phcr.yaml
